# \# Алатау — автопутешествия по Северному Кавказу

# 

# Адаптивный сайт‑лендинг для личного гида и водителя по Кавказу.  

# Проект рассказывает о маршрутах, республиках и легендах гор, с возможностью связаться напрямую.

# 

# \## 🚗 Возможности

# \- Адаптивная верстка для всех устройств  

# \- Мобильное меню  

# \- Слайдер маршрутов (Swiper.js)  

# \- Плавная прокрутка и анимации  

# \- Защита контактов от спам‑ботов  

# \- Форма обратной связи

# 

# \## 🛠️ Технологии

# HTML, CSS, JavaScript, Swiper.js
